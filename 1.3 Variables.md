```ruby
     type = map

     }

     type = set(string)

}
```

### How would you fetch the value of the key called slow from the variable called hard_drive in a terraform configuration?

Use key matching.

key = slow

So here we fetch the value for the key called slow from the map as:

var.hard_drive["slow"]
